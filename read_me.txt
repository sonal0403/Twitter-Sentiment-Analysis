Note1: This application requires Twitter apps account and Alchemy API key.

1. In working directory create a file called reqimports.py
2. Create twitter apps account and a new appliaction.
3. Using twitter apps account populate access token, access key, customer secret and customer key in reqimports file.
4. Register for a free language processing api key @ alchemyapi.com
5. Assign alchemy api key to alap variable of reqimports.


Note2: Run realtimeplot.py only after tweet_sentiment_result.txt is generated in working directory.
